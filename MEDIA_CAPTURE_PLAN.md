# Media Capture Plan

## Viewport
**1440 x 900** (standard desktop)

## Capture Routes

### Dashboard Views
```
/dashboard?preset=balanced&media=1
/dashboard?preset=texture&media=1
/dashboard?preset=barrier&media=1
```

### Results Views
```
/results?preset=balanced&media=1
/results?preset=texture&media=1
/results?preset=barrier&media=1
```

## Capture Rules
- No scroll, capture above-the-fold content only
- No mouse cursor visible
- No browser chrome (use fullscreen mode or crop)
- Wait 2 seconds after page load for image/font loading
- Consistent time of day (use same system time for timeline consistency)

## File Naming
```
softmaxxer-dashboard-balanced.png
softmaxxer-dashboard-texture.png
softmaxxer-dashboard-barrier.png
softmaxxer-results-balanced.png
softmaxxer-results-texture.png
softmaxxer-results-barrier.png
```

## Quality
- PNG format, 2x resolution (2880 x 1800 actual)
- Export at 1440 x 900 for final delivery
